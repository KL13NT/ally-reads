This options is used to chop paragraphs into shorter sentences.

Enabling this will allow the extension to apply the styles set using these selector.

You have to refresh open webpages after changing the settings. All settings are auto-saved.

Note that this extension currently supports only HTML paragraph tags. For contact you can check my social links on my website

For bug reports and questions you can open issues on GitHub
